# OptiGSM - Examples

Basic examples showing how OptiGSM communicates with Android devices via ADB.

## adb-devices.js

Lists all connected ADB devices and reads basic device properties.

```bash
node examples/adb-devices.js
```

**Requirements:**
- Node.js 18+
- ADB installed and in PATH
- Android device with USB Debugging enabled

## About OptiGSM
