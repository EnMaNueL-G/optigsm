Drivers USB — Instalar / Descargar

Drivers necesarios para que Windows reconozca cualquier dispositivo Android, tablet o TV Box por USB.

Universal ADB Driver
Funciona con TODOS los Android: teléfonos, tablets, TV Box. El más completo.
Google USB Driver
Oficial Google — Pixel, Nexus, Android One. También incluye ADB.
Samsung USB Driver
Para todos los Samsung: Galaxy S/A/M/Tab. Necesario para Odin/Heimdall.
Qualcomm HS-USB
Driver QHSUSB para modo EDL (9008). Xiaomi, OnePlus, Motorola QC.
MTK VCOM Driver
MediaTek preloader / BROM mode. Necesario para mtkclient.
Instalar via winget
Instala ADB + platform-tools oficiales de Google en segundos.
Después de instalar drivers, desconecta y reconecta el dispositivo. Activa "Depuración USB" en Opciones de desarrollador.

Instalar dependencias

pip install mtkclient edlclient pyserial pyusb
scoop install adb heimdall (Windows)
choco install adb heimdall-flash (alternativa)
winget install Google.PlatformTools (ADB oficial)

Activar ADB en el teléfono

  1. Ajustes → Acerca del teléfono → Número de compilación (pulsa 7 veces)
  2. Ajustes → Opciones de desarrollador → Depuración USB: ON
  3. Conecta el cable y acepta el aviso de depuración en el móvil

Modos especiales

MTK BROM: Apaga → Vol- + conectar USB
Samsung Download: Vol- + Home + Power (o Vol-+Vol+ en S10+)
Qualcomm EDL: Vol+ + Vol- al conectar (o test point)
Fastboot: Vol- + Power (varía por modelo)
Recovery: Vol+ + Power (Samsung), Vol- + Power (Pixel)

IMEI en Android 10+

Desde Android 10, ADB no puede leer el IMEI sin privilegios de sistema.
Opciones:
• Root: activa root y usa Modo Avanzado → IMEI
• MTK: usa MTK/Preloader → Leer NVRAM (sin root)
• QC: usa Qualcomm → Leer EFS (modo EDL)
• Samsung: se puede leer con Heimdall en Download Mode

Heimdall — cómo incorporarlo

Heimdall es open-source (MIT) y se puede incrustar sin instalación externa.
Opción 1 (recomendada): Descarga heimdall.exe y colócalo en la carpeta tools/ del proyecto — OptiGSM lo detectará automáticamente.
Opción 2: scoop install heimdall o winget install Heimdall
Opción 3: Configura la ruta en Ajustes → Rutas de herramientas.

📺 TV Box Android — compatibilidad

¿Funciona OptiGSM con TV Box? Sí. Los TV Box corren Android y son 100% compatibles con ADB.

Conectar via WiFi (más común en TV Box):
1. En el TV Box: Ajustes → Opciones de desarrollador → Depuración ADB por red: ON
2. Anota la IP del TV Box (Ajustes → Red)
3. En OptiGSM → Inicio → "Conectar WiFi" → escribe la IP
4. O en Terminal ADB: adb connect 192.168.1.X:5555

Qué puedes hacer con un TV Box:
• Instalar/desinstalar APKs (launcher, apps)
• Deshabilitar bloatware del fabricante
• Cambiar la densidad de pantalla (wm density)
• Instalar launchers alternativos (Wolf Launcher, ATV, etc.)
• Flash firmware via Fastboot (si el dispositivo lo soporta)
• Root via Magisk si el bootloader está desbloqueado
• Leer y exportar logs del sistema

Nota: El Terminal ADB → categoría "TV Box / Android TV" incluye comandos específicos para estos dispositivos.
